## macOS
- **Cocoa API:** Object-Oriented-API
- **CocoaTouch(Framework):** AppKit의 iOS 용 버전 (CocoaTouch ⊃ UIKit)
- **AppKit:** 네이티브 macOS 애플리케이션 개발을 위한 toolkit, 그래픽 인터페이스를 만드는 클래스들을 제공
